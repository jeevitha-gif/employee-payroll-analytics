
# Employee Payroll Analytics Project

## 📊 Project Overview
This project analyzes employee salary and workforce data using MySQL, Excel, and Power BI.

## 🛠 Tools Used
- MySQL (Data Import)
- Excel (Data Cleaning & Transformation)
- Power BI (Data Visualization)

## 🔹 Steps Involved
1. Imported CSV data into MySQL
2. Cleaned and transformed data in Excel
3. Applied formulas (IF, SUMIF, COUNTIF, INDEX-MATCH)
4. Built Power BI dashboard

## 📈 Key Insights
- Salary distribution
- Employees by city
- Gender analysis
- Experience level insights

## 📁 Files Included
- Dataset (CSV/Excel)
- SQL Queries
- Power BI Dashboard (.pbix)

## 🚀 Outcome
Developed an end-to-end data analysis project for business insights.
